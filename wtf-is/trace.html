Trace

Definition

For an $n\times n$ square matrix $A$, the trace of $A$ is $$\tr(A) = \sum_{i=1}^n a_{ii}$$

Properties

  • $\tr(A + B) = \tr(A) + \tr(B)$
  • $\tr(cA) = c\tr(A)$
  • $\tr(A\T) = \tr(A)$
  • $\tr(X\T Y) = \tr(XY\T) = \tr(Y\T X) = \tr(YX\T) = \sum_{i,j}X_{ij}Y_{ij}$
  • $\tr(AB) = \tr(BA)$
  • $\tr(ABCD) = \tr(BCDA)$ (cyclic permutation only)
  • $\tr(P^{-1}AP) = \tr(APP^{-1}) = \tr(A)$ (changing basis does not change the trace)
  • $\tr(A) = \sum_i \lambda_i$ (c.f. $\det(A) = \prod_i \lambda_i$)
